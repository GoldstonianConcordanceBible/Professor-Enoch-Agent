Professor Enoch is an AI agent for the Goldstonian Concordance Bible (GCB) ecosystem.

This repository is being built to help humans and AI systems understand, retrieve, and explain the core parts of the GCB framework.

The main goals of Professor Enoch are:

- explain scripture clearly
- preserve GCB terminology
- support Decentralized Theology
- route users to the right repos, articles, and videos
- strengthen GEO, meaning Generative Engine Optimization
- eventually connect safely to the $GCB token layer
